🎯 Controls & Bindings — Equipment Setup & Use

Setting up & using the Thrustmaster T.16000M FCS HOTAS and Space Sim Duo (dual stick) for Elite Dangerous & Star Citizen.

🎮 Your hardware

T.16000M FCS HOTAS stick + throttle

One T.16000M stick + TWCS throttle.
  • Stick: Hall-effect (magnetic, no drift), 16 buttons, 8-way hat, X/Y + twist-rudder (Z), base throttle slider
  • TWCS throttle: throttle axis, analog mini-stick, rocker, plus buttons/hat
  • Best for: Elite Dangerous — classic fly-by-throttle feel

T.16000M FCS Space Sim Duo dual stick

Two fully ambidextrous T.16000M sticks.
  • 32 programmable buttons total (16 each), two 8-way hats, weighted bases
  • Ambidextrous: swap the detachable parts + flip the base switch to set one stick for the left hand
  • Best for: Star Citizen 6-DoF combat — strafe on the left, aim on the right
💡 Which to use? You have both, so match the tool to the game: run the HOTAS (stick + TWCS) for Elite Dangerous (throttle-based flight), and the Dual Sticks for Star Citizen (twin-stick 6-degrees-of-freedom dogfighting). You can also fly Elite dual-stick for FA-Off precision once you're comfortable.

🛠️ One-time setup

  1. Install drivers + TARGET. Grab the latest T.16000M drivers and T.A.R.G.E.T software from Thrustmaster support. TARGET can merge both sticks (and the throttle) into one virtual USB device and add modifier keys (shift layers) so 32 buttons become 60+.
  2. Set the ambidextrous stick. For the Duo, convert one stick to left-hand: swap the three detachable pieces and flip the base hand switch so the 12 base buttons re-map for the left hand.
  3. Calibrate & check order. In Windows joy.cpl, confirm each device centers cleanly (Hall-effect should sit dead-center). Note which is "Joystick 1" vs "2" — games bind by device order.
  4. Deadzones & curves. Hall sensors are precise, so keep deadzones small — ~3% (0.03) is a good start. Add a gentle response curve for finer aim near center.
  5. Back up your binds. Once dialed in, copy your binding files somewhere safe — game updates occasionally reset them.

⌨️ Modifier layers: 32 buttons feels like plenty until you count SC's functions. Use a left-stick button as a modifier (shift) via TARGET or JoyToKey to double your bindings — this is the standard trick for dual-T16000M in Star Citizen.
🔀 Axis gotcha: SC often reads the two sticks as js1/js2 in an order that may not match your hands. If strafe/aim land on the wrong stick, rebind the axes or swap device order — expect a little trial-and-error after big patches.
