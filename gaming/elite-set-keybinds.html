⌨️ See & Set Your Keybinds (& fix Disembark)

How to view every current binding, change one, and why "press E" did nothing.
🟠 Career Hub 🥾 On-Foot ⌨️ Keybind Reference 🔧 See & Set Binds
🆘 Stuck in the SRV this second? Open your Role Panel (bottom panel, default 3) → Self-Destruct (hold), or Recall/Dismiss Ship and board it. You respawn in your ship; the SRV rebuilds free. Works with no keybinds at all.

❓ Why "press E" did nothing

Pressing a key does nothing unless that action is actually assigned to it. Your preset is Thrustmaster T.16000M FCS HOTAS — a flight-only preset that leaves Disembark (and most on-foot actions) blank. So E wasn't "Disembark" — Disembark had no key. You have to open Controls and assign it first. Also: you must be fully stopped, and if E is already used by something else the game will flag a conflict (red) — pick a free key.

👀 See your current binds

In-game (per action)
  1. Options ▸ Controls. Each action row shows its currently assigned key/button on the right.
  2. An empty box = unbound (that's what Disembark looks like right now).
  3. Scroll the category list; expand a category with its + to see all its actions and current keys.
See ALL binds on one sheet (best way)
  1. Open Windows Explorer and paste this into the address bar:
    %LOCALAPPDATA%\Frontier Developments\Elite Dangerous\Options\Bindings
  2. Grab your active .binds file (the custom one, e.g. Custom.4.0.binds).
  3. Go to EDRefCard.info → upload that file → it draws a printable reference card of every current binding across ship, SRV & on-foot.

🔧 Set or change a bind

  1. Options ▸ Controls.
  2. Find the action (for walking out of the SRV: scroll to the ON FOOT category, expand +, find Disembark).
  3. Click the empty box next to it → press the key you want (e.g. E or G).
  4. If it turns red / warns of a conflict, that key is already used — pick a different free key.
  5. Do the same for Embark (to get back in). Back out — it saves automatically.
