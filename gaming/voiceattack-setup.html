⚠️ Read this first — it's the thing that breaks most setups

VoiceAttack works by pressing keyboard keys. It cannot press your joystick buttons.

Most of your Elite controls are on the stick and throttle — landing gear is Stick Button 14, boost is Throttle Button 3, supercruise is Throttle Button 5. VoiceAttack can't trigger any of those as they stand.

The fix: Elite lets every action have a Primary AND a Secondary binding. Leave your joystick bind as Primary, and add a keyboard key as Secondary. Your HOTAS keeps working exactly as it does now — you just give VoiceAttack a key it can press.

💡 Use keys you'd never press by accident. Bind these secondaries to the Numpad or F-keys. They stay out of the way of your existing keys (W A S D E Q R V F H N M L U Z Shift Ctrl Space Alt are all in use), so there are no conflicts.

1️⃣ Install & microphone

  1. Install VoiceAttack and launch it.
  2. Set your microphone in Windows Sound settings as the default input device, then run Windows Speech Recognition training once.This single step is the biggest accuracy improvement you can make — it teaches Windows your voice, and VoiceAttack uses that engine.
  3. In VoiceAttack, pick your mic from the device dropdown.
  4. Set a listening mode. Push-to-talk (hold a key/button to speak) avoids false triggers; always-listening is more immersive.If you have a spare throttle button, push-to-talk on it is ideal.

2️⃣ Give Elite keys for VoiceAttack to press

  1. In Elite: Options ▸ Controls.
  2. Find an action you want by voice (e.g. Landing Gear).
  3. Click the SECONDARY box — the second slot, not the one showing your joystick button.Leaving Primary alone is what protects your HOTAS setup.
  4. Press the key you're assigning (e.g. Numpad 1).
  5. Repeat for each action in the table below. Back out — Elite saves automatically.
⚠️ Do this while Elite is closed or at the main menu, and don't edit the binds file by hand at the same time — the game rewrites it on exit.

4️⃣ Create the command in VoiceAttack

  1. In VoiceAttack, click the pencil / Edit Profile icon.
  2. Click New Command.
  3. In "When I say", type your phrase — e.g. landing gear.You can add alternatives separated by semicolons, e.g. landing gear;gear down;deploy gear — so any of them work.
  4. Add an action: Key/Mouse ▸ Press and Release Key(s).
  5. Choose the key you set as Elite's Secondary (e.g. Numpad 1). Leave duration at the default short press.
  6. OK ▸ OK ▸ Done. Repeat per command.
✅ Test it: with Elite in the foreground, say the phrase. VoiceAttack's log shows it heard you, and the ship should respond. If VoiceAttack hears it but nothing happens in-game, the Secondary bind is missing or on a different key.

5️⃣ Optional: voice packs

Packs like HCS Voice Packs add an AI co-pilot personality plus hundreds of pre-built Elite commands — you import a profile instead of building each command. They still rely on the same principle: they press keys, so your Elite bindings must match what the pack expects. Most packs ship a reference sheet of the keys they use.

VoiceAttack hears me, nothing happens in EliteThe action has no keyboard bind. Add the key as a Secondary in Elite's controls.
It mishears / triggers randomlyRun Windows Speech Recognition training, use push-to-talk, and make phrases 2+ words (single words misfire).
Works on desktop, not in-gameRun VoiceAttack as Administrator — Elite may ignore input from a lower-privilege app.
Command fires twiceTwo commands share a similar phrase, or key duration is too long. Shorten the press.
