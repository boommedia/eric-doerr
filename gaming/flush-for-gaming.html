🎮 Flush for Gaming

Free up RAM & CPU after a long VS Code + AI session, then launch your game on a clean machine.

▶ Run the Flush ⚙ Open the PowerShell script
Before you click: the flush force-closes VS Code — save your work first. Clicking Run the Flush may show a browser download/security prompt (it's just a local .bat); choose Keep / Open, then approve the Windows UAC prompt so it can empty standby RAM.

🧹 What it does, in order

  1. Elevates to Admin — relaunches with a UAC prompt (needed to empty standby RAM).
  2. Shows top 10 RAM users BEFORE — the "before" snapshot.
  3. Closes VS Code & AI/dev helpers — Code, node, esbuild, rg, tsserver, Copilot, language servers, vsls-agent.
  4. Clears temp files — wipes %TEMP%.
  5. Empties standby RAM — via RAMMap / EmptyStandbyList if present in C:\Tools (skipped otherwise).
  6. Restarts Explorer — clears accumulated explorer.exe bloat.
  7. Shows top 10 RAM users AFTER — the "after" snapshot so you can see what you freed.

⚡ Enable the standby-RAM flush (optional)

Step 5 only runs if one of these Sysinternals tools sits in C:\Tools. Grab either one, drop the .exe there, and the flush gets stronger automatically:

RAMMap64.exe ↗ Sysinternals ↗
